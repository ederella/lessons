В Java каждый класс или интерфейс, как правило сохраняется в отдельном файле .class.
Классы объединяются пакеты. 
Пакеты объединяются в модули (начиная с Java 9). 
Наследование возможно для классов. 
Класс из одного пакета или модуля может расширять класс из другого пакета или модуля. 
Подключение происходит с помощью ключевого слова import.
